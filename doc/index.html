Readable Human Format

Getting started

This is a quick guide to get started with generating documentation with https://readable.Red The library is still in very infancy stage (for example sub-title is not yet supported for markdown whereas it is for html) so it will become much richer in the future.
Currently it is capable of generating markdown and html page like this very page or for example:

https://redlang.red/source

Pre-requisite

Install Redlang and VSCode Extension

Follow instruction on this page:

https://readable.red/#next

Generating Markdown

Step 1:

Write a source in Readable Human Format, for example for this page:


Title: {Readable.red}
Sub-Title: {Getting started}

Intro: [
.text: {This is a quick guide to get started with generating documentation with https://readable.Red
The library is still in very infancy stage so it will become much richer in the future.


Currently it is capable of generating markdown and html page like this very page or for example:
}

.links: [
    https://Redlang.Red/source
]
]

Pre-requisite: [
    .title: {Installation}
    .sub-title: {Follow instruction on this page:}
    .link: https://readable.red/#next
    .image: %redlang.png
]

Markdown: [
    
    .title: {Generating Markdown}
    .sub-title: {Step 1:}
    .text: {Write a source in Readable Human Format, for example for this page:}    
]

.code: {
Title: {Readable.red}
Sub-Title: {Getting started}    

Intro: [
.text: {This is a quick guide to get started with generating documentation with https://readable.Red
The library is still in very infancy stage so it will become much richer in the future.


Currently it is capable of generating markdown and html page like this very page or for example:
}

.links: [
    https://Redlang.Red/source
]
]

Markdown: [
    
    .title: {Generating Markdown}
    .sub-title: {Step 1:}
    .text: {Write a source in Readable Human Format, for example for this page:}    
]
.code: {
    etc...
}
}

Step 2:

Save the readable document as for example:

content.read.red

Create a file named for example:

gen-markdown.red

inside that file, type this code:


do https://readable.red
article: load %content.read.red
markdown-gen/output %article.md

In Vscode, right-click to get menu and choose option:

Run Red Script
