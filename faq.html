Can I split a long file into multiple files

Readable is made for large documentation or book, so it is part of its spec. To do so, split content into multiple sections wrapped with [] and put each of them in a file, then adapt the build script, like this for example:


do https://readable.red

article: copy []
foreach file read %parts/ [
    append article load rejoin [%parts/ file]
]
html-gen/output %getting-started.html
